# budget_shinydashboard

An interactive financial management tool built with *R* and *shinydashboard*. This application allows users to track expenses, visualize spending patterns, and manage personal or household budgets through a clean, intuitive interface.

For the time being the dashboard includes the following three tabs:

## 1) General overview
The full time series of income and expenditures is shown above.
In the barplot below, the black dotted line indicates the average revenue by month, while the green dotted line is customisable and represents the desired target. 

![GeneralOverview](https://github.com/cateinthesky/budget_shinydashboard/blob/main/shiny-d1.gif)

## 2) Detailed analysis based on a selected month
In the dropdown menu it is possible to select the desired month.
Below, the donut chart shows the percentage of income that was spent in the selected month. On the left there is the detail of expenditures for the selected month.

![GeneralOverview](https://github.com/cateinthesky/budget_shinydashboard/blob/main/shiny-d2.gif)
